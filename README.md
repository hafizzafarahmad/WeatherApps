# WeatherApps

Racangan aplikasi Prakiraan Cuaca saat ini dan 5 hari kedepan.
Aplikasi ini menggunakan:
1. Bahasa pemrograman Java
2. OpenWeatherMap API
3. Fragment
4. Gson
5. rxjava
6. rxandroid
7. Retrofit
8. Picasso
